Подготовка Ignore

Добавьте нужные фабрики в список ниже. Сайт сформирует команды для запуска addIgnore.py.

Можно подобрать сразу несколько фабрик и применить их одним нажатием.

    Список Ignore пока пуст. Добавьте фабрики через кнопку выше.

    После выбора нажмите «Применить», чтобы записать фабрики в список Ignore.

    Нажмите «Применить», чтобы записать выбранные фабрики в Ignore.

      Выберите фабрики
